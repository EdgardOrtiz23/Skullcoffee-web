# Skullcoffee-web
Nombre del Proyecto: Skull Coffee | Café Arábico de Especialidad
¿Qué es y Para qué Sirve?
Skull Coffee es un sitio web de marketing local que promueve y vende café de especialidad de alta calidad de Nicaragua, cumpliendo con los requisitos de un proyecto web colaborativo. El objetivo principal es establecer una presencia digital clara y profesional para la marca, permitiendo a los clientes locales y potenciales conocer los procesos de tostado únicos (Honey, Lavado, Natural) y realizar pedidos en línea.

Solución que Ofrece
Esta página web soluciona varios problemas clave para la marca:
Visibilidad y Marca: Ofrece una plataforma moderna y temática para presentar la historia de la marca y sus productos de manera atractiva.
Organización del Catálogo: Permite a los clientes diferenciar entre los distintos procesos de café (Honey, Lavado, Natural), facilitando la elección del producto perfecto.
Ventas Locales: Implementa una funcionalidad de carrito de compras simple para gestionar pedidos y cotizaciones, centralizando el proceso de venta.

Colaboración Profesional: Utiliza GitHub Actions para asegurar la calidad del código y automatizar la publicación, garantizando un flujo de trabajo de desarrollo eficiente y sin errores.

Tecnologías Utilizadas
Este proyecto fue desarrollado bajo un flujo de trabajo colaborativo riguroso  utilizando las siguientes herramientas:

HTML: Uso de HTML semántico para una estructura web accesible y bien organizada.

CSS: CSS personalizado para el diseño temático de café, incluyendo un sistema de variables.

JavaScript: Implementación de JavaScript funcional para la lógica del carrito de compras interactivo.

Control de Versiones y Colaboración:

GitHub: Alojamiento del repositorio, uso de ramas (main y desarrollo), y gestión de tareas con Issues.

GitHub Actions: Automatización de tareas clave como la validación de HTML y el despliegue automático a producción.

GitHub Pages: Plataforma de hosting gratuita para la publicación del sitio web final.
